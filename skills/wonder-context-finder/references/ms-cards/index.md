# MS Cards Index

This index lists all MS Cards extracted from the Master Data V2.0 Use Cases documents.

## Overview

MS Cards document business requirements in the format `MSxx-yy`:
- **MS01 series**: Item type management
- **MS02 series**: Item list and search
- **MS03 series**: Item creation
- **MS04 series**: Item detail views
- **MS05 series**: Item detail UI cards (Smallwares, BOM, Version, etc.)
- **MS06 series**: BOM and component operations
- **MS08 series**: Major operations (Version, Dormant, Status changes)
- **MS09 series**: Import/Export operations
- **MS10-MS12 series**: Additional configurations
- **MS13 series**: Complex features (Change ticket, Customization)
- **MS15 series**: Additional operations (Pack size, Cost)
- **MS16-MS18 series**: Extended features
- **MS23 series**: Specialized operations

**Source**: Master Data V2.0 Use Cases documents
**Location**: `assets/source-docs/` (within this skill)
- `034-Master Data V2.0 Use Cases-1.docx`
- `034-Master Data V2.0 Use Cases-2.docx`

> **Note**: For complete MS Card details, read the original Word documents in `assets/source-docs/`
